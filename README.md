# Personal portfolio

Project developed with:

- React
- Typescritp
- Tailwind

Deployed in: https://pablo-webdeveloper.com

# Run project

To run the project you just need to:

1. Install the dependencies

```
npm install
```

2. Run project with docker

```
docker compose up
```

3. Alternatively, you can use:

```
npm run start
```

To run the linter:

```
npm run lint
```
